# Lab-3

This repository contains a **CSC 326 (Data Structures)** lab project written in C++.

## Overview
The program models a customer service workflow using two core data structures:
- **Queue** for waiting customers (FIFO)
- **Stack** for missed customers (LIFO)

The project reads customer records from a file, separates them by state, and serves them according to a fixed policy.

## Concepts Practiced
- Linked-node implementation
- Stack and queue design
- File input handling
- Program flow and service-order logic
- Time aggregation and formatted output

This repository reflects course lab work and data structures practice completed as part of CSC 326.
